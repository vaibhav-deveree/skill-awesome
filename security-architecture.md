

# Universal Security Architecture Skill

## Security First Principle

Security reviews are mandatory.

---

# Authentication

Review:

* Session-based
* Token-based

Document decision.

---

# Authorization

Review:

* Roles
* Permissions
* Ownership

Never trust frontend validation.

---

# OWASP Review

Mandatory:

* Access Control
* Injection
* Authentication Failures
* SSRF
* Security Misconfiguration

---

# Secrets Management

Never:

* Hardcode secrets
* Commit secrets

Use secure secret storage.

---

# Logging Review

Never log:

* Passwords
* Tokens
* Secrets

---

# Security Agents

1. Security Architect
2. Security Engineer
3. Reliability Engineer
4. Principal Engineer

---

# Final Checklist

✓ OWASP reviewed

✓ Secrets reviewed

✓ Authorization reviewed

✓ Authentication reviewed
